Projects

Protein DataBase Analyzer:

The program reads a PDB file containing the position and size of molecules. The purpose of the program is to differentiate voids and cavities in the protein structure. Cavities are empty spaces in a structure that are completely enclosed instead of voids which are open to the outside. The program uses a discretization algorithm to search for these areas and creates a new PDB file that marks the location of cavities. The program can be set to different resolution and cavity size to customize the users search.
